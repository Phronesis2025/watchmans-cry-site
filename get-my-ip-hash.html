Step 2: Add to Environment Variable

Add your IP hash to the EXCLUDED_IP_HASHES environment variable in Vercel:

  1. Go to your Vercel project dashboard
  2. Navigate to Settings â†’ Environment Variables
  3. Add a new variable:
    • Name: EXCLUDED_IP_HASHES
    • Value: Your IP hash (or comma-separated list of hashes)
    • Environment: Production, Preview, Development (select all)
  4. Redeploy your application for changes to take effect

Note: If you have multiple IP addresses (home, office, mobile), you can add them all separated by commas.

Alternative: Client-Side Opt-Out

You can also disable tracking entirely in your browser using localStorage:

  1. Open your browser's developer console (F12)
  2. Run this command to opt out: 
    localStorage.setItem('watchmans_cry_opt_out', 'true')
  3. To opt back in, run: 
    localStorage.removeItem('watchmans_cry_opt_out')

Note: This only works for the browser where you run the command. It won't affect other devices or browsers.
